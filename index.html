Visual Studio Code, also commonly referred to as VS Code, is a source-code editor made by Microsoft with the Electron Framework, for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git. To setup VS code download ownload and install VS Code; open a folder already downloaded on your device, ceate a new file and start working on your project.
